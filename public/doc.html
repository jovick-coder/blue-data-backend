Welcome BlueData API documentation

Register Api

POST https://blue-data-api.herokuapp.com/api/user/register
this request is a post request which will register a new user

Request Body

{
"email": "name@email.com",
"fullName": "name surname ",
"userName": "UserName",
"phoneNumber": 0987654321,
"password": "*********",
"privilege": 1
}
Note: This Account is for testing propose only

Response

200
{ ok: true }
400
{ ok: false, message: "email is required", }
email is empty or not given
400
{ ok: false, message: "phoneNumber is required", }
phoneNumber is empty or not given
400
{ ok: false, message: "password is required", }
password is empty or not given
400
{ ok: false, message: "privilege is required", }
privilege is empty or not given
500
{ ok: false, message: "Server error", error: error, }
Error normally caused by Duplicated email or password.. email or password used already exist in the database

Login Api

POST https://blue-data-api.herokuapp.com/api/user/login
this request is a post request which will log a new user

Request Body

{
"email": "name@email.com",
"password": "*********"
}
Note: This Account is for testing propose only

Response

200
{
"ok": true,
"token": "jwtToken",
"privilege": 1
}
404
{ ok: false, message: "User Not Found", }
Account not Found in the database
400
{ ok: false, message: "email is required", }
email is empty or not given
400
{ ok: false, message: "password is required", }
password is empty or not given
500
{ ok: false, message: 'Server Error', error:error }
This is a Server error
